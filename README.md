# HDB Resales Dashboard

A full-stack web application for analyzing HDB Resales data.

## Development Workflow

This project uses an AI-Agentic workflow. 
Please refer to the **[Workflow Guide](.agent/workflows/README.md)** for instructions on how to plan, implement, and maintain features.

---

### Key Directories

*   **`src/`**: Application source code.
*   **`docs/`**: Project documentation (PSD, Plans, Architecture).
*   **`.agent/workflows/`**: Workflow definitions for the AI agent.
*   **`.github/prompts/`**: Source prompts for the workflows.
